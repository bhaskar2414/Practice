## 🚀 Getting Started

### Prerequisites

Ensure you have the following installed:

- **Node.js** (Recommended: Latest LTS version)
- **npm** or **yarn**

### Installation

1. **Clone the repository:**

```bash
git clone https://github.com/bhaskar2414/Practice.git
cd Practice
```
